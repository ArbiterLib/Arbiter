1 # Arbiter
2 
3 Arbiter is a C library which implements the basic behaviors needed by any decentralized dependency manager or package manager, without being coupled to any particular platform or purpose, so that more specific tools can be built on top.
4 
5 For example, Arbiter could be used to implement a package manager for:
6 
7  * **Development libraries**, similar to [npm](http://npmjs.com), [CocoaPods](https://cocoapods.org), [Cargo](https://github.com/rust-lang/cargo), etc.
8  * **Installable tools**, similar to [Homebrew](http://brew.sh)
9
